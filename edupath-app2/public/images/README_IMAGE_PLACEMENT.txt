INSTRUCTIONS FOR LANDING PAGE BACKGROUND IMAGE
===============================================

To use your building image as the background for the landing page:

1. Place your image file in this directory: public/images/
2. Name it one of the following:
   - baganga-building.jpg
   - baganga-building.png
   - building.jpg
   - building.png

3. Supported formats: JPG, PNG

4. Recommended image specifications:
   - Resolution: 1920x1080 or higher
   - File size: Under 2MB for best performance
   - Format: JPG for photos, PNG for graphics

5. The landing page will automatically detect and use the image.
   If no image is found, it will use a gradient background instead.

6. After placing the image, refresh the landing page to see it.

The image will be displayed as a full-screen background with a dark overlay
and centered "Welcome to Baganga Online Enrollment" text.
